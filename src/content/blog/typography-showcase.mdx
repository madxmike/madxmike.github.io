### Code Blocks

Here's a JavaScript function:

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

// Usage
const result = fibonacci(10);
console.log(`The 10th Fibonacci number is: ${result}`);
```

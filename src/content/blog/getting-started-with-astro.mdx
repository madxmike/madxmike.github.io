# Getting Started with Astro

Astro is a modern web framework for building fast, content-focused websites. It's designed to deliver exceptional performance by shipping zero JavaScript by default.

## Why Astro?

- **Fast by default**: Ships zero JavaScript unless you need it
- **Islands architecture**: Add interactivity where needed
- **Framework agnostic**: Use React, Vue, Svelte, or any framework

## Key Features

1. Component Islands
2. Server-first API design
3. Zero JS by default
4. Built-in optimizations

# Building Better Social Media Embeds

When someone shares your blog post on social media, you want it to look great. This post covers how to implement Open Graph and Twitter Card meta tags to create rich, engaging previews.

## What Are Social Media Embeds?

Social media embeds are the rich previews you see when someone shares a link on platforms like Twitter, LinkedIn, or Facebook. They typically include:

- A title
- A description
- An image
- The site name

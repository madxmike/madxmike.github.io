# Building Interactive Components

Astro's islands architecture allows you to add interactivity exactly where you need it, keeping the rest of your site fast and static.

## What are Islands?

Islands are interactive UI components on an otherwise static page. They hydrate independently, allowing for optimal performance.

## Adding Interactivity

You can use any framework you like:

- React
- Vue
- Svelte
- Solid
